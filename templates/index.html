Employee Turnover Prediction Website

Dataset

The dataset used for the training of this website contains 14999 rows and 9 question variable to predict the target wheather the employee will stay or quit. The 9 variables used for the prediction are satisfaction level (in %), last evaluation i.e., the satisfaction level last time this observation was made (in %), number of projects the employee is currently working on, Average monthly hours employee works in the company, number of years the employee has already worked in the company, wheather the employee has been involved in a work accident before, wheather the employee has received any promotion in last five years, the department in which the employee works in and the salary range i.e., low, medium or high.

Training Methods

We used several different different machine learning algorithms, for the prediction of the employee will wheather stay or quit the company.In total we used 7 different machine learning algorithms for the purpose of prediction.The name of these machine learning algorithms is as follows, Decision Tree Classifier, K Nearest Neighbors, Logistic Regression, Multi Layer Perceptron, Gaussian Naive Bayes, Random Forest Classifier and Support Vector Machine.The different accuracy score which we observed from all these machine learning algorithms is as follows:-

Method Score
Decision Tree Classifier 98.3
K Nearest Neighbors 99.2
Logistic Regression 79.3
Multi Layer Perceptron 89.9
Gaussian Naive Bayes 93.8
Random Forest Classifier 64.8
Support Vector Machine 90.4
